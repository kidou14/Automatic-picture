OneClick · 新装饰元素候选

精选 4 种新装饰类型

基于设计师建议的20个方向，筛选出改动最小、效果最显著的 4 种装饰元素。 每种均以最适合它的配色模式渲染，展示在现有 SVG decoration 框架内的实际效果。

Type 01
Noise Grain
方案05 暗夜极光
10行代码 neon-dark · cool-dark
neon-dark + noise
cool-dark + thin border
实现方式
新增 isNoiseGrain: true decoration type，在 decorationSvg() 里加一个分支
SVG <feTurbulence> filter + feBlend mode="overlay"，全画布覆盖，不影响渲染速度
neon-dark / cool-dark 模式下自动附加，opacity 0.3–0.55 随机
同时在 dark 系模式下，以 30% 概率为 slide 加 1px 细边框（rgba accent, 0.2）
Type 02
Center Pulse
方案03 数字脉冲
15行代码 neon-dark · dark
neon-green-dark
cool-dark + indigo pulse
实现方式
新增 isCenterPulse: true decoration type，origin 固定在 画布底部 2/3 处（手机底座位置）
与现有 isRing 的区别：ring 从四角扩散，center-pulse 从中心向外，呈雷达感
同时新增 neon-green-dark colorMode（bg #040e07，accent #00ff88），专属配色
ring 间距和 opacity 按 指数衰减，最外圈 opacity 约 0.06，视觉上非常干净
Type 03
Scanlines
方案15 复古未来
3行代码 所有模式通用
retro-future + scanlines
dark + ultra-subtle scanlines
实现方式
在 buildSlideHtml() 里，dark 系模式下以 40% 概率在 slide 最外层添加一个 scanline div
只有一行 CSS：repeating-linear-gradient(transparent 0 3px, rgba(0,0,0,0.12) 3px 4px)，零性能损耗
强度随 colorMode 变化：retro 模式用 0.18 opacity，其他 dark 模式用 0.06–0.10，subtle 但有质感
同时新增 retro-dark colorMode（紫红 + 青 neon，用于左侧极致效果），80s 风格专属
Type 04
Hairline Grid
方案06/16 精密网格
20行代码 light · pastel · dark
light + hairline grid
dark + hairline grid + axis
实现方式
新增 isHairlineGrid: true，SVG <pattern> 细网格（0.35px stroke，24px spacing）+ radial mask 淡出边缘
额外绘制 2 条轴线（垂直 + 水平中线，opacity 稍高），+ 4 个十字定位符号在四角位置
light / pastel 模式下，网格颜色用 accent 色（12% opacity）；dark 模式用 accent（10%），视觉上"隐约存在"
这个装饰覆盖全画布而不是角落定位，在 decorationSvg() 里是一个特殊全幅类型
