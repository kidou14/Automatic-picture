# Logo 自动生成规则文档

> 对应代码：`scripts/screenshot-server.js` → `suggestAppLogoQwen()` 函数
> 想调整任何内容，直接对着这个文件说即可。

---

## 一、整体流程

1. 用户在 preview.html 输入 App URL 和 App 名称
2. 服务端从目标页面提取品牌色（`pageColors`）和正文摘要（`bodyText`）
3. 从「主体形象词库」随机抽一个 `tradingSubject`
4. 从「设计风格库」随机抽一个 `style`（或由前端指定 styleIndex）
5. 拼成完整 prompt 发给 Qwen（DashScope `wanx2.1-t2i-turbo`）
6. 返回图片 base64，展示在页面 Logo 区域

---

## 二、完整 Prompt 结构

```
请生成一张 512×512 正方形品牌徽章图片。画面结构极其简单：一个纯色或渐变的背景色块铺满整个画布，一个几何图形符号居中叠于其上。就这两层，没有其他任何东西。

【背景层】纯色或线性渐变色，直角填满画布的每一个角落，不存在任何圆角、白边、透明区域。

【主体层】单一几何图形符号，居中放置，占画面60%～70%，直接浮于背景色之上。

【绝对禁止出现的形状】在上述两层之外，画面中不得出现任何多余的形状，尤其是：带圆角的矩形色块、正方形色块、深色底板、浅色底板、白色衬底、投影、光晕、任何将主体"托起来"的底座形状。背景就是背景，没有第二层背景。

【设计风格】几何抽象，参考 Spotify、Discord、Apple 等品牌的标志设计：形状简洁大胆，色彩对比强烈，无写实细节，无材质纹理，无叙事场景。

【主体形象】将「{tradingSubject}」转化为简洁的几何符号：保留其核心形态特征，去除所有写实细节，只保留最关键的轮廓线或色块。

【配色】优先绿色、蓝色、金色、青色、深紫、橙色；避免大面积红色（金融语境中红色代表亏损）。

品牌名称：{appName}
{descHint}
风格要求：{style}
{colorHint}

输出要求：
- 几何符号轮廓粗壮清晰，缩小到小尺寸仍可辨认
- 整体干净，只有背景+符号两个视觉元素
- 禁止出现任何文字
```

### 动态变量说明

| 变量 | 来源 | 说明 |
|---|---|---|
| `{tradingSubject}` | 主体形象词库随机抽取 | 见第三节 |
| `{appName}` | 用户输入 | App 名称 |
| `{descHint}` | 用户输入 desc 参数 / 页面正文前200字 | 格式："应用主题描述：xxx。" |
| `{style}` | 设计风格库随机抽取 / 前端指定 styleIndex | 见第四节 |
| `{colorHint}` | 从目标页面提取品牌色 | 格式："背景色调参考品牌色：#xxx、#xxx，" / 无品牌色时为空 |

## 四、设计风格库（6种，gallery页可全部预览）

> gallery 页会把6种风格全部展示出来供选择，preview 页随机抽取一种。
> styleIndex 0~5 对应下列顺序。

| Index | 风格名 | 描述 |
|---|---|---|
| 0 | 极简单色 | 极简扁平风格，纯色饱和背景（深蓝/深绿/深紫任选其一）加纯白色几何图形主体，仅2色，强对比，无渐变无阴影，参考 Apple、Twitter X 图标的克制与力量感 |
| 1 | 渐变扁平 | 鲜艳多彩渐变背景（参考 Spotify 绿、Instagram 暖渐变或 Airbnb 珊瑚红），白色极简几何主体，图形简洁轮廓清晰，背景渐变方向为45°斜向，整体年轻有活力 |
| 2 | 深色轮廓 | 深黑或深海军蓝背景，主体为品牌色（青绿/电光蓝/金色）描边的几何线条轮廓，带有轻微外发光效果，参考 Discord、Linear 的深色图标风格，科技感强 |
| 3 | 撞色几何 | 高饱和双色撞色构图，背景与主体颜色均为高饱和度（如电光蓝配金黄、深紫配青绿），图形用几何切割方式分割画面，参考 Figma、Dribbble 的大胆配色风格 |
| 4 | 磨砂渐变 | 柔和多色磨砂渐变背景（紫→蓝→青，或金→橙→黄），渐变过渡自然，白色半透明几何主体叠于其上，整体质感柔润现代，参考 iOS 16+ 系统图标风格 |
| 5 | 等轴测几何 | 等轴测（isometric）视角的几何立方体或多面体构成，3种颜色的色块拼合出立体感，无材质纹理，纯色块构成，参考 Notion、Pitch 的现代极简3D图标风格 |

## 五、生成方案

当前仅保留 **Qwen** 和 **DALL·E** 两种方案，Claude SVG 已移除。

| 方案 | 模型 | 尺寸 | 超时 | 说明 |
|---|---|---|---|---|
| **Qwen**（默认）| `qwen-image-2.0`（DashScope multimodal-generation）| 512×512 | `DALLE_LOGO_AI_TIMEOUT_MS`（默认 50s）| 支持 styleIndex 指定风格、交易词库随机注入 |
| **DALL·E** | `dall-e-3`（OpenAI）| 1024×1024 | `DALLE_LOGO_AI_TIMEOUT_MS`（默认 50s）| 返回 base64 PNG，无风格指定 |

- **页面品牌色提取超时**：`APP_LOGO_PAGE_TIMEOUT_MS`（默认 8 秒）
- **前端调用超时**：`DALLE_LOGO_TIMEOUT_MS`（默认 70 秒，含网络往返）
