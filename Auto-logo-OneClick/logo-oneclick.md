# Logo 生成 Prompt

修改后需同步到 `scripts/logo-server.js` 的 `PROMPT_STYLE`、`STYLES` 和 `buildPrompt` 函数。

---

## 生成逻辑

每次生成随机选 1 套风格 + 随机抽 1 个主题词，颜色规则两套共用。

---

## 风格 1：macOS 磨砂玻璃质感

```
请生成一个精致的512×512，macOS风格的磨砂质感的logo，极简主义，配色年轻活泼。

应用名称：{appName}

设计风格：
磨砂玻璃感，超高对比度渐变, 图形占画布80%面积，轻微厚度和细腻投影，半透明，C4D高清渲染品质，清脆干净的边缘，没有文字，没有字母，没有水印,禁止下方衬底

主题：{theme}

颜色要求：最多2种颜色搭配，图标主体与背景高对比度，明媚活泼
```

---

## 风格 2：渐变抽象

```
请生成一个精致的512×512，现代未来主义的抽象logo，配色年轻活泼。

应用名称：{appName}

设计风格：
流线型几何图形或有机曲线,充满活力的霓虹多色渐变(蓝紫粉绿橙虹彩色系),呈现流动感、半透明光泽和发光边缘,微妙的3D立体感,纯色背景,极简主义设计,无文字,高清矢量插画，没有文字，没有字母，没有水印，禁止下方衬底

主题：{theme}

颜色要求：1-3种颜色搭配，图标主体与背景高对比度,高级专业
```
